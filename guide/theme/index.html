Theme

Ant Design Web3 is developed based on Ant Design, and you can set the theme of Ant Design Web3 by configuring Ant Design's theme.

Style Switching

Ant Design Web3 has built-in support for dark mode, which inherits the configuration from Ant Design.

Custom Design Token

Customize Ant Design's Design Token

You can fully customize Ant Design's Design Token, which we will inherit and use in Ant Design Web3. If you find any unsupported or incorrectly used Design Tokens, please submit an issue to let us know.

We provide a theme editor where you can modify Design Tokens online and see the effects.

Customize More Design Tokens for Ant Design Web3

In addition to Ant Design's Design Tokens, Ant Design Web3 also offers some additional Design Tokens that you can customize. The Design Tokens for components are continuously being supplemented, and you can view the supported Design Tokens on the corresponding component documentation pages. If you find any unsupported or incorrectly used Design Tokens, please submit an issue to let us know.
