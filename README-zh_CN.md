## 特性

- 📦 开箱即用的体验
- 🎨 灵活的主题定制能力
- 🔌 兼容不同的链

## 安装

```shell
npm i antd @ant-design/web3 --save
```

## 使用

```ts
import { Address } from '@ant-design/web3';

export default () => {
  return <Address address="0x1234567890123456789012345678901234567890" />;
};
```
